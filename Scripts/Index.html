Welcome to my Genshin Impact database! It has everything you need to learn about the characters, weapons, artifacts, and bosses in miHoYo's 2020 open-world action role-playing game, Genshin Impact.

The Characters page contains a complete and updated list of all the playable characters that have been released as of the Genshin Impact v3.0 update.

The Weapons page includes every one, two, three, four, and five star weapon as of the v3.0 update.

The Artifact set page lists every artifact set in v3.0.

The Trounce Domains page includes information relating to the Trounce bosses released as of v3.0. This includes Dvalin, Andrius, Childe, Azhdaha, La Signora, and the Raiden Shogun.
